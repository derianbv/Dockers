Pasos para descargar una imagen de DockerHub: 

0. Verificar si tengo docker: 

docker --version

1. Si hago: docker images, confirmo las imagenes que tengo descargadas (no debería tener ninguna).

2. docker pull hello-world

Docker busca esa imagen llamada hello-world en el registro oficial de imágenes Docker público llamado Docker Hub (hub.docker.com) y la descarga a tu máquina local si no la tienes ya.

2.2 ahora "docker images" muestra la imagen descargada:

REPOSITORY    TAG       IMAGE ID       CREATED       SIZE
hello-world   latest    d2c94e258dcb   2 years ago   13.3kB

3. Correr la imagen: "docker run hello-world": 

"Hello from Docker!

 4. The Docker daemon streamed that output to the Docker client, which sent it
